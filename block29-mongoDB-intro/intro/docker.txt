Criar servidor mongo pc

1) docker pull mongo 

***How to use this image

         docker run --name container-name -d mongo:tag

... where some-mongo is the name you want to assign to your container and tag is the tag specifying the MongoDB version you want (-d desaclopado do terminal, mongo(imagem))

2)           docker run --name container-name -p 27017:27017 -d mongo

(-p: mapeamento de portas) -> mongo escuta interno(dentro do container): 27017
                           -> mongo atende fora(minha maquina)):27017
